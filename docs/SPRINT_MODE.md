# Sprint Mode

Sprint Mode groups multiple task workspaces into an iteration.

## Trigger examples

```text
/dev-baseline 创建迭代 v0.3.9 sprint-1
创建迭代
规划本轮迭代
```

## Workspace

```text
docs/sprints/YYYYMMDD-vX.Y.Z-sprint-slug/
```

## Purpose

A sprint workspace records:

- sprint goal
- included task workspaces
- task priorities
- capacity and rough effort
- risks
- sprint progress
- sprint review
- retrospective
