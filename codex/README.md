# Codex Adapter

Codex uses the canonical Dev Baseline skill package from `../skill`.

Do not add duplicated skill files, agents, references, or templates here. Update `../skill` first, then let `scripts/install-dev-baseline.sh` generate Codex personal installs or project overlays from that package.

Useful commands:

```bash
bash scripts/install-dev-baseline.sh codex
bash scripts/install-dev-baseline.sh codex-project /path/to/project
```
