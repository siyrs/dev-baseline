# Dev Baseline Sprint Skill

Use this skill to create or review a sprint workspace.

## Trigger examples

- `/dev-baseline-sprint create v0.3.9 sprint-1`
- `创建迭代`
- `规划本轮迭代`

## Command

```bash
bash shared/scripts/create-sprint-workspace.sh <version> <sprint-name>
```

- Sprint goal:

- Capacity:
